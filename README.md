# Renewable Energy Share Analysis Tool

This repository contains code and resources to analyze and visualize the 'Renewable Energy Share by Year (2013-2024)' dataset. The tool is designed to provide stakeholders with actionable insights into the trends and growth of renewable energy adoption in Abu Dhabi over the past decade.

## Features

- **Interactive Visualizations**: Generate charts and graphs to visualize trends and patterns in renewable energy adoption.
- **Summary Statistics**: Calculate and export summary statistics for further analysis.
- **Automated Updates**: Ensure access to the latest data through regular updates.
- **User-Friendly Interface**: Navigate the platform easily with search and filter functionalities.

## Requirements

- Python 3.7 or higher
- Pandas
- Matplotlib

## Installation

1. Clone the repository:
   bash
   git clone https://github.com/yourusername/renewable-energy-share-analysis.git
   

2. Navigate to the project directory:
   bash
   cd renewable-energy-share-analysis
   

3. Install the required Python packages:
   bash
   pip install -r requirements.txt
   

## Usage

1. Place the dataset file (`Renewable_Energy_Share_2013-2024.csv`) in the project directory.
2. Run the analysis script:
   bash
   python analysis_tool.py
   
3. The script will generate visualizations and summary statistics, which will be saved as files in the output directory.

## Results

- `Renewable_Energy_Share_Trend.png`: A line chart showing the trend of renewable energy share over the years.
- `Renewable_Energy_Share_Summary.csv`: A CSV file containing summary statistics for the dataset.
